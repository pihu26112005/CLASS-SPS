# Smart Parking System

## Project Description
A web-based Smart Parking System that detects available slots, prevents double bookings, and tracks parking history in real-time. Built with Python (Flask) and JavaScript.

## Dependencies
* Python 3.8+
* Flask (`pip install flask`)
* Flask-CORS (`pip install flask-cors`)

## Installation & Running
1. Navigate to the `Code` directory.
2. Install dependencies:
   ```bash
   pip install -r Code/Backend/requirements.txt
3. Run the backend
    ```bash
    python Code/Backend/app.py
4. Now the Parking System can be seen fully functional on web. (open Code/Frontend/index.html in browser)
